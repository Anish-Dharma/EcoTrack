# 🌱 EcoTrack - Carbon Footprint Tracker

**EcoTrack** is a lightweight, responsive web app that helps individuals measure and reduce their carbon footprint. It focuses on two key contributors: **transportation** and **electricity consumption**, and offers real-time insights to support a more sustainable lifestyle.

![EcoTrack Banner]([Logo.png](https://i.pinimg.com/474x/79/3e/28/793e28d7cb88ca029409a3e16de578c9.jpg))

---

## 🔍 Features

- 🚗 **Transport Emission Calculator**  
  Log daily travel distances and view your CO₂ emissions from vehicles.

- ⚡ **Electricity Usage Analyzer**  
  Enter your electricity consumption (in kWh) and see the environmental impact.

- 📊 **Real-Time Emission Calculation**  
  Accurate CO₂ estimates based on science-backed emission factors.

- 🎯 **Simple & Intuitive UI**  
  Clean design with clear inputs, ideal for all age groups.

- 🔐 **Privacy First**  
  No login, no data collection—everything runs on the client side.

## 🚀 Getting Started

1. **Clone the repository**  
   ```bash
   git clone https://github.com/your-username/EcoTrack.git
   cd EcoTrack
   ```

2. **Open the app**  
   Double-click `index.html` or open it in your preferred browser.

3. **Track your emissions**  
   Use the “Start Tracking” button to go to `CFtPrt.html` and enter your data.
